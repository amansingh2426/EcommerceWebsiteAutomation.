🛒 Ecommerce Website Automation Testing
📌 Project Overview
This project focuses on automating the testing of an e-commerce web application using modern QA practices. It includes end-to-end testing of critical user workflows such as login, product browsing, cart operations, and checkout.

The goal of this project is to ensure application reliability, functionality, and user experience through structured testing.

🚀 Features Tested
User Authentication (Login / Signup)
Product Search & Filtering
Add to Cart / Remove from Cart
Checkout Process
UI Validation & Navigation Flows
Form Validation & Error Handling
🧪 Testing Type
Functional Testing
End-to-End Testing (E2E)
Regression Testing
UI Testing
🛠️ Tech Stack
Cypress (Automation Testing)
JavaScript
Node.js
