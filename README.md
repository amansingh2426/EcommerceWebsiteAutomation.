# 🛒 Ecommerce Website Automation Testing

## 📌 Project Overview
This project focuses on automating the testing of an e-commerce web application using **Selenium WebDriver with Java**. It covers end-to-end testing of critical user workflows such as login, product browsing, filtering, cart operations, checkout, and logout.

The goal is to ensure **application reliability, functionality, and seamless user experience** through structured automation testing.

---

## 🚀 Features Tested
- User Authentication (Login / Logout)  
- Product Search & Filtering  
- Add to Cart / Remove from Cart  
- Checkout Process  
- UI Validation & Navigation Flows  
- Form Validation & Error Handling  

---

## 🧪 Test Cases

### ✅ TestCase01: Verify Login Functionality
- Validates user login with correct credentials  
- Ensures successful navigation after login  

### 🔍 TestCase02: Applying Filter Functionality
- Applies product filters (e.g., price sorting)  
- Verifies products are displayed in ascending order of price  

### 🛒 TestCase03: Add to Cart Functionality
- Adds products to the shopping cart  
- Verifies cart update and item presence  

### 💳 TestCase04: Checkout Functionality
- Proceeds through checkout steps  
- Validates order placement flow  

### 🔐 TestCase05: Verify Logout Functionality
- Logs out the user from the application  
- Verifies successful logout and redirection

## ⚙️ Installation & Setup

### Clone the Repository
```bash
git clone https://github.com/amansingh2426/EcommerceWebsiteAutomation.git
cd EcommerceWebsiteAutomation
```

### Install Dependencies
```bash
mvn clean install
```

### Run Tests
```bash
mvn test
```
